# Doodle Icons Reference Guide

A comprehensive guide to help developers and AI assistants select the most appropriate icons for specific use cases. Each icon includes visual descriptions, semantic meaning, and practical application examples.

## How to Use This Guide

**For AI Assistants:** When suggesting icons for a user's website or application:
1. Identify the context (e.g., "banking website", "contact page", "checkout flow")
2. Look at the relevant category section
3. Scan the "Use Cases" column for matching scenarios
4. Suggest 2-3 icon options that fit the context

**For Developers:** 
- Browse by category to find available icons
- Read the "Represents" column to understand semantic meaning
- Check "Use Cases" for implementation ideas

- Numbered variants: `Bag2Icon`, `Dashboard3Icon`, `Sun4Icon`
